Security Policy
Supported Versions

JavaKids Lite is in active prototype development.
Since this is not a production environment, there are no formal version releases yet.
All updates apply to the current version only.

Reporting a Vulnerability

If you discover a security concern in this prototype, you can report it using GitHub Issues.
Please describe the issue clearly and include reproduction steps if possible.
As this project is developed by a single founder, response time may vary, but all valid security reports will be reviewed.
